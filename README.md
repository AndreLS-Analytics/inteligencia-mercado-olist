# 📊 Inteligência de Mercado: Análise de Performance E-commerce (Olist)

Este projeto aplica conceitos de **Inteligência de Mercado** e **Data Analytics** para extrair insights estratégicos de uma operação real. 

## 🎯 Objetivo do Projeto
Transformar dados brutos em decisões inteligentes, focando em:
*   **Performance Logística**: Análise de prazos e eficiência de frete.
*   **Inteligência de Vendas**: Identificação de categorias rentáveis e comportamento do consumidor.
*   **Geração de Insights**: Dashboards para suporte à tomada de decisão executiva.

## 🛠️ Stack Tecnológica
*   **SQL**: Extração e manipulação de grandes volumes de dados.
*   **Python**: Limpeza de dados e análise exploratória.
*   **Power BI**: Visualização de dados e criação de KPIs.
*   **Excel (VBA)**: Automação de relatórios complementares.

## 🚀 Status do Projeto
> 🏗️ **Em desenvolvimento**: Atualmente na fase de modelagem de dados e estruturação das queries SQL.
